# Simple Stack

Simple Stack is a full-stack framework for creating web apps at hyper-speed. It is based on
Ruby on Rails and Stimulus Reflex.

# Quick start

```
    git clone https://github.com/johnreitano/simple-stack.git
    cd simple-stack
    ./simple-stack.sh ~/my-project
    cd /path/to/my-project
    rails server -p
    open http://localhost:3000
```
* 

# Primary Components
    * Front end
        * HTML/CSS
        * Flexbox
        * Bootstrap
        * ERB
        * Turbolinks
        * StimulusJS (via Stimulus Reflex)
    * Backend
        * Ruby on Rails
        * Highlighted gems
            * Stimulus Reflex
            * Devise
            * Pundit
            * Sidekiq
        * Elasticsearch
        * Postgres
        * Docker Compose
        * Redis
        * Heroku
        * Docker
    * Devtools
        * Minitest
        * Rubocop        
        * VS Code + editor plugins
        * Git
        * Github Issues
